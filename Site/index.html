02. Projects

Some of my most recent projects.

Forty Servants

HTML / CSS / JAVASCRIPT

The Forty Servants is a divination deck web application made for desktop, and is based upon The Forty Servants by Tommie Kelly. Complete with Youtube video links and realtime information about each card. Choose your spread and shuffle the decks.

Visit Github

Apple.com Clone

HTML / CSS / JAVASCRIPT

This is a recreation of the apple.com landing page from late December 2020. Since I could not gain access to Apples font, I took some liberties with the font sizes and margins to achieve a look that was as close to the original as possible.

Visit Github

Cebeko Accounting

HTML / CSS / JAVASCRIPT / RESPONSIVE / BOOTSTRAP 4

This is a standard business site that I've made for an Accounting firm called Cebeko Redovisning AB. It was made to include certain preferences from the customer, such as GDPR modal, SRF newswatch and login to their servers.

Visit Github

Badass Burgers

HTML / CSS / JAVASCRIPT / RESPONSIVE

Badass Burgers is a fictitious hamburger restaurant. The site was made to mimmick some of the already existing hamburger restaurants in the area, and features all relevant things for a restaurant landing page.

Visit Github
